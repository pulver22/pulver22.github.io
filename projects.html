My Projects

I was and I'm involved in several projects. The relevant are almost all hosted on my GitHub repository.

Horse Fever: card game based on horse races, in which every player can bet on one horse and cheat the race to try to win money.

PDDL: source code written during the "Cognitive Robotics" course on my first master year at Politecnico di Milano.

Reversing Engineering: material used during tutorial lessons taken by ToH(Tower of Hanoy) computer security group of Politecnico di Milano.

Travel Dream: webapp developed during the "Software Engineering 2" course that allows user to buy packages that include flight, hotel and escursions.

Gamma correction: small script that allow to change the gamma of a random image in PGM (portable grayscale mal) format, using openMP and openMPI technologies.

3D TDLAS Gas Sensor Simulator: Implementing in ROS the simulator of a Tunable Diode Laser Absortion Spectoscopy sensor for remote gas detention.

MCDM online exploration: The online algorithm will calculate the path by combining the utility values over a set of multiple criteria, exploiting the concepts of Multi Criteria Decision Making (MCDM) and Choquet Fuzzy Integral in order to generate a single global utility value for each candidate location, and thus selecting the best one.

Next-Best-Smell algorithm: ROS package implementing the MCDM framework for remote gas detection using a mobile robot equipped with a remote gas sensor.
